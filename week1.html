Week 1: HTML Basics

1. Document Structure

The bare bones of every HTML page starts with a <!DOCTYPE html> declaration. It tells the browser to render your page according to HTML5 specifications. HTML4 had different doctypes for various modes, but HTML5 simplified this to a single declaration.

After the doctype, you have the <html> root element, which contains a <head> for about the page and a <body> for content. Everything inside <body> is shown inside the main browser window.

The head usually contains a <title> tag for the page title shown in the browser tab. And zou will usually find a <meta charset="UTF-8"> tag to specify character encoding, and other meta tags for viewport settings, SEO, and more. Links to CSS stylesheets and JavaScript files are also placed in the head.

Attributes are used to provide additional information about an element. They are always included in the opening tag and come in name/value pairs like name="value". Common global attributes include id, class, and style. We will learn more about attributes in the next session when we talk about CSS. 

<!DOCTYPE html> 
<html lang="en">
  <head>
    <title>My First HTML Page</title>
    <meta charset="UTF-8">
    <meta name="description" content="This is a simple demo page." >
    <meta name="keywords" content="HTML, CSS, JavaScript">
    <meta name="author" content="Qianxun Chen">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="main.css" rel="stylesheet" />
    <script src="main.js"/></script/>
  </head>
  <body>
    <p class="text" id="greet" >Hello world</p>
  </body>
</html>

2. Headings + Paragraphs

HTML has six levels of headings: <h1> to <h6>, with <h1> being the most important one . You can use headings to structure your content hierarchically. For regular text, use <p> for paragraphs. By default, there are usually some margins around headings and paragraphs to create spacing.

<hr> tag is used to create this horizontal line that separats sections of content visually. You can also format your text further with <b> for bold and <i> for italic. Notice that these changes are purely visual and they don't influence the content at all.

3. White Spaces and Line Breaks

When the browser comes across two or more spaces next to each other, it only display one space. Similarly if it comes across a line break, it treats that as a single space too. This is known as white space collapsing. 

<p>This is a line with     extra spaces. But you don't see them.
Same with line-breaks.</p>

This is a line with extra spaces. But you don't see them. Same with line-breaks.

To create a line break without starting a new paragraph, one can use the <br> tag. This is useful for poems, addresses, or any content where you want to control line breaks without having to start a new paragraph. 

<p>This is a little poem<br>with lines<br>and line breaks</p>

4. Links

Links are created with <a> tags. The href attribute are used to specify the URL. There are absolute URLs (starting with http:// or https://) that point to external sites, and relative URLs that point to other pages within your site. If you want a page to open in a new tab, add the target="_blank" attribute to your link. 

<a href="https://www.github.com" target="_blank">Visit github.com with absolute url</a> |
<a href="index.html">Go back to home page with relative url</a>
Visit github.com with absolute url | Go back to home page with relative url

A special mailto link can be created with mailto: in the href, which will open the user's default email client to send an email to the specified address. 

<a href="mailto:qianxun.chen@leuphana.de">Send me an email</a>
Send me an email

You can also go to a particular location of a page by using an anchor link. The targeted element must have an id attribute with the corresponding name.

<a href="#lists">Jump to lists section</a>
Jump to lists section

5. Media

The <img> tag stands for image.

  • src attribute specifies the image URL
  • title attribute is the text that is displayed with a tooltip on hover
  • alt attribute provides alternative text for people who can't see the image
  • You can also set the width and height attributes to control the size of the image.

<img width="400" title="web developer meme" src="https://cdn.getshifter.co/8a02b8322c8339db67673735bae4923b9ee1696e/uploads/2024/03/web-development.jpeg" alt="I heard you want to be a web developer, here are a few devices to test your site">

You can add <audio> and <video> content to your HTML. Add the controls attribute to show playback controls. 

<audio controls>
  <source src="media/sound_effect.mp3" type="audio/mp3">
</audio>
<video controls width="400">
  <source src="media/sample.mp4" type="video/mp4">
</video>

6. Lists

<ul> stands for unordered lists and <ol> for ordered lists. Both use <li> for list items. Unordered lists typically display bullet points, while ordered lists display numbers or letters before each item. You can also nest lists inside each other for sub-lists.

<!-- Unordered Lists -->
<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<!-- Ordered Lists -->
<ol>
  <li>First step</li>
  <li>Second step</li>
</ol>

<!-- Nested Lists -->
<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrot</li>
      <li>Broccoli</li>
    </ul>
  </li>

7. Groups

Some elements will always appear to start on a new line in the browser window. These are known as block-level elements. Ex: <h1>, <p> and <ul>. Other elements can appear within a line of text and are called inline elements. Ex: <a>, <span>, and <img>.

<div> is a block-level container for grouping elements, while <span> is an inline container for styling parts of text. We will use these tags a lot when we learn CSS to style our page, as they allow us to apply styles to specific sections of our content without affecting the overall structure. 

<div style="background:#f0f0f0; padding:1rem;">
    This is a div container with a gray background.
</div>
<div style="background:orange; padding:1rem;">
    This is a div container with an orange background.
</div>

8. Tables

Inside a <table>, use <thead> for the header row and <th> for header cells. Use <tbody> for the body rows, <tr> for each table row, and <td> for regular cells. By default, tables have borders and padding, but you can customize their appearance with CSS. 

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alice</td>
      <td>20</td>
    </tr>
  </tbody>
</table>

10. Semantic Tags

Semantic tags are HTML elements that clearly define their content's purpose to both the browser and the developers.

Instead of <div> or <span> tags one can use <header>, <nav>, <main>, <article>, <section>, <aside>, <footer> in HTML5 to define different elements of a page. 

<header>...</header>
<main>...</main>
<footer>...</footer>

<em>Emphasized text</em>
<blockquote>A famous quote</blockquote>

11.Extra Markups

Comments in HTML are written with <!-- comment here --> and are ignored by the browser. They are useful for leaving notes in your code or temporarily disabling sections of code during development.

<!-- This is a comment in HTML -->

iframes are like little windows to other web pages. Use <iframe> with a src to embed another page within your page. This is commonly used for embedding videos, maps, or other external content.

<iframe width="560" height="315" src="https://www.youtube.com/embed/fJWR7dBuc18?si=KZkVeOrzCiI5KX7K" 
title="YouTube video player" frameborder="0" allowfullscreen></iframe>

Some characters are reserved in HTML. If you use the less than (<) or greater than (>) signs in your HTML text, the browser might mix them with tags. To display these characters as text, you need to use HTML entities: &lt; for < and &gt; for >.Other common entities include &amp; for & and &quot; for double quotes ("). For a more complete list of HTML entities, check this page. 

&lt;div&gt;This will show as <div> in the browser&lt;/div&gt;
