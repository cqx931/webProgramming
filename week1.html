Week 1: HTML Basics

1. Document Structure

The bare bones of every HTML page starts with a <!DOCTYPE html> declaration, then an <html> root with <head> and <body>.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My Page</title>
  </head>
  <body>
    <p>Hello world</p>
  </body>
</html>

Rendered: Hello world

2. Headings + Paragraphs

Headings <h1> through <h6> structure content with decreasing size. Paragraphs use <p>.

<h4>Subheading</h4>
<h5>Subheading</h5>
<h6>Subheading</h6>

3. Links

<a> is for links, <img> embeds images. Always include href and alt for accessibility.

<a href="https://www.github.com" target="_blank">Visit github.com</a>
Visit github.com

4. Images

Use <img> tag for images. Always include alt for accessibility.

5. Lists (Ordered + Unordered)

Use <ul> for bullet lists and <ol> for numbered lists with <li> items.

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>First step</li>
  <li>Second step</li>
</ol>
  • Apple
  • Banana
  1. First step
  2. Second step

6. Tables

<table> with <thead>, <tbody>, rows (<tr>), headers (<th>), and cells (<td>).

<table>
  <thead>
    <tr>
      <th>Name</th>
      <th>Age</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alice</td>
      <td>30</td>
    </tr>
  </tbody>
</table>

6. Forms (Inputs and Buttons)

Use <form> elements for interactive user input. Fields include <input>, <textarea>, <select>, and <button>.

<form>
  <label>Name:<input type="text" name="name" placeholder="Your name"></label>
  <label>Message:<textarea name="msg" rows="3"></textarea></label>
  <button type="submit">Send</button>
</form>

7. Semantic Tags

Use meaningful structure tags for accessibility and SEO: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>.

<header>...</header>
<nav>...</nav>
<main>...</main>
<footer>...</footer>
